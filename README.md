## 📌 Featured Academic Projects

### 🤖 PILIA — Introductory Platform for AI Literacy

Educational and gamified web platform designed to introduce Artificial Intelligence concepts to secondary school students.

**Main features:**

- Modular learning path
- Quizzes and progress tracking
- Sequential content unlocking
- Ranking system
- Daily challenge
- Final certificate

**Technologies:**  
`C#` `ASP.NET Core MVC` `Entity Framework Core` `SQL Server` `Razor Views` `Bootstrap`

### 🌍 One Health Distributed System

Distributed system for collecting, processing and analysing urban environmental data through sensors, gateways and a central server.

**Main features:**

- TCP socket communication
- RabbitMQ Publish/Subscribe
- JSON-RPC services
- Processing of JSON, CSV and XML data
- SQLite data persistence
- Measurement deduplication
- Concurrent access synchronisation
- Web dashboard with historical data, statistics and charts

**Technologies:**  
`C#` `Python` `RabbitMQ` `SQLite` `JSON-RPC` `TCP/IP`

### 🚗 DCarMarketplace — Used Vehicle Marketplace

Web platform for buying, selling and managing used vehicles, with separate roles and functionalities for buyers, sellers and administrators.

**Main features:**

- Buyer, seller and administrator profiles
- Vehicle listings
- Search and filtering
- Reservations and visit scheduling
- User and advertisement management
- Administrative back office

**Technologies:**  
`C#` `ASP.NET Core MVC` `Entity Framework Core` `ASP.NET Identity` `SQL Server` `Razor Views` `Bootstrap`
